wcc.search Installation
-----------------------

To install wcc.search using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.search`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.search

* Re-run buildout, e.g. with:

    $ ./bin/buildout
